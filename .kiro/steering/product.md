# Product Overview

This is an automated beat generation system that uses AI to produce instrumental music tracks at scale. The system integrates multiple AI APIs (Gemini, OpenAI, Suno) to create diverse beats across various genres, styles, and moods.

## Core Functionality

- Automatically generates beats every 15 minutes based on a predefined catalog
- Manages multiple Suno API keys to optimize free quota usage
- Creates beats across 20+ categories including lo-fi, trap, cinematic, afrobeats, and more
- Generates metadata (names, tags, descriptions) for each beat using AI
- Stores all beats, prompts, and metadata in MySQL for reuse and analysis

## Target Use Cases

- Study/productivity music (lo-fi, ambient)
- Rap/freestyle beats (trap, drill, pluggnb)
- Content creation (YouTube, TikTok, vlogs)
- Commercial/corporate background music
- Film/documentary soundtracks
- Meditation and sleep audio

## Key Features

- Multi-API key rotation for continuous production
- AI-enhanced prompt generation for quality and variety
- Comprehensive metadata for easy beat discovery
- REST API for integration with other applications
- Automated scheduling with 24-hour template rotation
